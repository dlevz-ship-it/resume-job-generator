How it works

1. User input

The user enters a target role, their experience, and a job description into the web form.

2. JavaScript logic

The app uses JavaScript to split the text, find repeated skill keywords, and create resume-style bullet points.

3. Browser storage

The Save Result button stores the latest generated bullets in localStorage, creating simple client-side data storage.
