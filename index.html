How it works

1. User input

The user enters a role, location, experience, and a job description into the web forms.

2. JavaScript logic

The app builds job-search URLs, scans for skill keywords, creates tailored resume bullets, and calculates a match score.

3. Browser storage

The Save Result button stores the latest generated bullets in localStorage, creating simple client-side data storage.
